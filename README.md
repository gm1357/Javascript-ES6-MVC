# Javascript-ES6-MVC

Repositório com projeto desenvolvido seguindo os cursos de Javascript Avançado da Alura

## Instalação

 - Entre em /server
 - `npm install`
 - `npm run build`
 - `npm start` para iniciar a api em `localhost:3000`
 - Entre em /client
 - `npm install`
 - `npm start` para iniciar a aplicação em `localhost:8080`
